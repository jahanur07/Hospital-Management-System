# Hospital-Management-System

Full-stack web application developed using Python Django

Manages patient records, doctor details, and appointments

Implements role-based access for Admin, Doctors, and Patients

Uses a relational database with proper SQL design and normalization

Supports appointment scheduling and medical record tracking

Ensures data security and integrity

Developed following Software Development Life Cycle (SDLC)

Strengthened skills in backend development, frontend integration, and database management
